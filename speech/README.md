# Speech

## SpeechWithAI.py

Eine erste version der Kommunikation mit der AI. Es benutzt speech to text, anschliessend ein API-Key, **welcher selbst noch generiert und eingefügt werden muss**, zum Schluss dann noch text to speech. Das text to speech war hier noch sehr unoptimiert, jedoch wurde dies dann bei der nächsten version behoben.

## betterVoice.py

Hier wurde die Stimme vorallem optimiert. Es wurde eine Menschlichere Stimme genommen und verständlicher gemacht.

Es werden relativ viele Libaries gebraucht. PyCharm kann viele davon selbst automatisch installieren, jedoch gibt es auch einige welche manuell [installiert](../dokumentations/PyCharm.md#packages-installation) werden.

## gesture.py

in dieser Datei hat es einige simple bewegungen, welche das reden "menschlicher" aussehen lassen sollte. Diese werden entsprechend der Länge des Textes zufällig ausgeführt.
